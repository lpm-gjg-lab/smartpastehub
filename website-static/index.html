Features

Everything your clipboard was missing.

20+ intelligent transforms run silently on every paste. Zero friction, maximum quality.

⚡

Auto-Clean on Paste

Hit Alt+Shift+V instead of Ctrl+V. SmartPasteHub intercepts the clipboard, runs the full transform pipeline, and pastes clean output into any app — silently.

🤖

AI Rewrite

Grammar fix, rephrase, formalize, summarize, translate and more. Bring your own API key.

📷

OCR Screenshot

Snap any screen area and extract text instantly. Press Ctrl+Alt+S — done.

📚

Multi-Copy & Paste Queue

Collect multiple clips and merge them. Enqueue items and paste one at a time.

👻

Ghost Write

Type prepared text into any window — even when direct paste is blocked or detected.

🔒

PII Security Scanner

Detects API keys, emails, phone numbers. Prompts to Mask & Paste before sensitive data leaks.

🧠

Smart Context Rules

Detects copy source and paste target. Applies different presets automatically (e.g. code passthrough to terminal).

🔤

Unicode Safety Net

Fixes mojibake, NBSP, smart quotes, zero-width characters, soft hyphens, CRLF, and fullwidth ASCII — automatically on every paste. Your text always lands clean.
